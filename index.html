Dev Bug

1. Ошибка JavaScript в консоли

Открой DevTools (F12) и перейди на вкладку Console, затем обнови страницу. В консоли появится красная ошибка nonExistingFunction is not defined.

Это пример того, как одна некорректная функция может ломать важный функционал или подгрузку контента.

SEO

2. Подозрительные мета‑теги

Посмотри исходный код страницы (Ctrl+U) или вкладку Elements в DevTools:

  • Мета‑тег robots со значением noindex,nofollow — поисковик не будет индексировать страницу.
  • Канонический URL указывает на другую страницу: https://example.com/wrong-page.
SEO Dev

3. Лишние скрипты и ресурсы

Открой вкладку Network в DevTools и обнови страницу:

  • Обрати внимание, что загружается внешний скрипт heavy-analytics.js.
  • Посмотри общее количество запросов и размер загружаемых ресурсов.
Dev

4. Мини‑код для проверки meta и canonical

Во вкладке Console попробуй выполнить такой код: 

const robots = document.querySelector('meta[name="robots"]');
const canonical = document.querySelector('link[rel="canonical"]');

console.log('robots:', robots && robots.content);
console.log('canonical:', canonical && canonical.href);

Так SEO‑специалист может быстро проверить важные теги без сторонних сервисов.
